Pràctica 1: Instal·lació i configuració d'un servidor DNS de reenviament

Objectius

Els objectius de la següent activitat són:

  • Instal·lar un servidor de noms.
  • Entendre els avantatges d'instal·lar un servidor de noms a la xarxa local.
  • Configurar un servidor de reenviament.
  • Configurar els clients de resolució de noms correctament.

Recursos necessaris

Per a poder realitzar la pràctica necessitaràs:

  • Un ordinador amb connexió a Internet o una maquina virtual amb Ubuntu Server.
  • El programari Bind.
  • La guia d'instal·lació:
    • https://help.ubuntu.com/16.04/serverguide/dns.html
    • https://help.ubuntu.com/community/BIND9ServerHowto
    • https://www.digitalocean.com/community/tutorials/how-to-configure-bind-as-a-private-network-dns-server-on-ubuntu-16-04

Enunciat

Instal·la i documenta el procés d'instal·lació del servei DNS sols com a servidor cau, és a dir, és dedica a respondre a les peticions i a emmagatzemar-les per a un ús posterior.

  1. Instal·la el servei.
  2. Quins són els arxiu de configuració necessaris?
  3. Configura el servidor com a servidor cau (cache).
  4. Comprova que funciona amb els comandaments dig, ping, nslookup i host.
  5. Documenta els resultats obtinguts.
  6. Fes una captura del registre on quede registrada una resolució de noms.
